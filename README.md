# Plant Moisture Sensor Workshop

This workshop takes you from a simple soil moisture sensor and an ESP32 to live readings on the shared plant dashboard.

You can go through the workshop in two different ways:

- [Guided path](./guided.md): step-by-step instructions with the exact wiring and code to use.
- [Unguided path](./unguided.md): the same workshop flow, but with challenge prompts, hints, and expandable answers when you get stuck.

The end goal for both paths is the same:

1. Get the board talking to Arduino IDE.
2. Read moisture values from the sensor.
3. Send those readings to https://codepub-nl.site/.

If you just want the most direct route, start with [guided.md](./guided.md). If you want to figure more of it out yourself first, start with [unguided.md](./unguided.md).
